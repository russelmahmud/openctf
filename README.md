# OpenCTF
Capture The Flag Open Platform

## Run locally
1. Run docker
```
$ docker-compose up
```
2. Visit site
```
http://localhost:8000
```
3. Admin panel
```
http://localhost:8000/admin/
username: admin
password: ctfadmin
```
